## 📋 Tasks Completed

### Task 3: Responsive Design
**Objective:** Make the webpage adapt to different screen sizes using CSS media queries.

**Implementation:**
- Created separate `styles.css` file and linked to HTML
- Added media queries for breakpoints at 768px (tablet) and 480px (mobile)
- **Desktop (>768px):** Profile image 150px x 150px, normal layout
- **Tablet (≤768px):** Image reduced to 120px, skills section stacked vertically
- **Mobile (≤480px):** Image reduced to 100px, button width 100%, reduced padding
- Adjusted font sizes and spacing for better readability on small screens
- Tested using Chrome Developer Tools for desktop, tablet, and mobile views

**Tech Used:** HTML5, CSS3, Media Queries

---

### Task 4: Interactive Button
**Objective:** Add JavaScript functionality to improve user engagement.

**Implementation:**
- Added a button element in `index.html` with id `colorBtn`
- Created separate `script.js` file for JavaScript code
- Used `addEventListener` to detect button clicks
- On click, background changes to random color from predefined array using `Math.random()`
- Added CSS transition for smooth color change effect
- Tested functionality on all devices to ensure it works properly

**Tech Used:** HTML5, CSS3, JavaScript, DOM Manipulation, Event Listeners
